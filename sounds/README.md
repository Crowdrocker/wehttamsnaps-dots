# WehttamSnaps Sound Files

Sound files are NOT included in this repo due to copyright.

## How to get them

Download from https://101soundboards.com

- **J.A.R.V.I.S.**: Search `jarvis-v1-paul-bettany-tts-computer-ai-voice`
- **iDroid**: Search `idroid-tts-computer-ai-voice`

## Where to put them

```
/usr/share/wehttamsnaps/sounds/jarvis/   ← JARVIS .mp3 files
/usr/share/wehttamsnaps/sounds/idroid/   ← iDroid .mp3 files
```

The installer creates these directories automatically.
See docs/INSTALL.md for the full expected filename list.
