# Photography Workflow on WehttamSnaps

## Workspace 5 - Photography
All photo editing apps launch here automatically

## Complete Pipeline

### 1. Import (DigiKam)
- Organize by date/event
- Add tags and metadata
- Initial culling

### 2. Develop (Darktable)
- RAW processing
- Color grading
- Exposure adjustments
- Lens corrections

### 3. Edit (GIMP)
- Advanced compositing
- Retouching
- Layer-based edits
- Export for web/print

### 4. Touch-up (Krita)
- Digital painting
- Advanced masking
- Texture work

### 5. Export (Custom script)
- Press Mod+Shift+E to export with sound
- Configurable presets for:
  - Wedding galleries
  - Social media (Instagram/Twitter/YouTube)
  - Print ready

## Quick Tips
- Import directly from camera: Thunar auto-mounts
- Color-accurate: Use DisplayCAL for calibration
- Batch processing: Darktable scripts available
