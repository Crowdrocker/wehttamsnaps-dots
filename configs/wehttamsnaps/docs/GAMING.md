# Gaming on WehttamSnaps Niri Setup

## Workspace Layout
- **Workspace 2**: Gaming (pre-configured)
- **Workspace 3**: Photography (when not gaming)
- **Workspace 6-7**: Additional gaming spaces

## Gaming Mode (Mod + G)
- Disables animations for maximum FPS
- Switches to iDroid sound system
- Enables GameMode CPU optimization
- Toggles performance profile
